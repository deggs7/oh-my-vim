oh-my-vim
=========

Make VIM as IDE(python, web)

### requirement

```
pip install flake8
```

### Install

```
git clone git@github.com:deggs7/oh-my-vim.git ~/.vim
cd ~/.vim
git submodule init
git submodule update
./install.sh
```

### Update

```
cd ~/.vim
git pull origin
./install.sh
```

### Shortcuts

* ,nt - NERDTree
